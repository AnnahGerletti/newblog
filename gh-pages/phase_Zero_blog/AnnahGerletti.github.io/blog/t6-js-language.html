JavaScript Language

June 8, 2017

JavaScript language or syntax is important because it makes up a set of rules the computer has to follow.
Below are the most common punctuation characters used in JavaScript.

Cases where javascript syntax is used

( ) = Parentheses

  1. Used to group expressions e.g.a = (4 + 3) * 2;
  2. Contains expressions e.g.(a < 10)
  3. Execute a function e.g.var result = doSomething()
  4. Executing a function with parameters e.g.
    function doSomething(help) {}
  5. Contains test logic for conditional execution statements i.e. loops and if/else if e.g
    if(a < 10) {doSomething};
    else if (a > 10) {doSomethingElse};
    for(var i = 0; i < 3; i++){};
  6. Provide Switch statement conditions e.g.
    switch ("weather forcast")
    case "rainy":
    console.log("Going to need an umbrella");
    break;
    case ...

[ ] = Brackets

  1. Defines an array e.g. myArray=[1,2,3,4];
  2. Accessing array elements using array index
    myArray = ["red","green","yellow"];
    console.log(myArray)[1]);
  3. Standard structure of bracket notation e.g.
    traffic ["lights"][2];

{ } = Braces

  1. Defines an object e.g.
    var colors ={
    "one": "blue",
    "red": 2,
    "flag": ["red","white","blue"]};
  2. Contain the logic of conditional expressions i.e. if/ else if statements and loops.
    var number = 3
    if (number < 4) {
    console.log("Number is lessthan 4")};
  3. Initializing a function
    function myDog(){
    return("woof")}

' ' = Single Quotes

  1. Identify strings e.g. alert( ‘Good morning!’);
  2. Accessing object properties with bracket notation e.g.name[‘Fred’]
  3. When you want quotes to appear inside a string. (e.g. for html attributes which we never do.)
    var string = “He is called ‘Johnny’”;

" " = Double Quotes

  1. Are used around strings which contain apostrophes. e.g.
    var answer = “That’s why”
  2. Used to access object properties with bracket notation. e.g.lunch[“sandwich”]
  3. Identifies a string e.g.var story = “Once upon a time.”
