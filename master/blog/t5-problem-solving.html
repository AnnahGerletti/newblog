Problem Solving

June 1, 2017

Through-out my life I have faced many challenges, using different techniques of problem solving to work them out. Since moving into my new apartment I have been facing a daily problem until recently.

By my bed sits a little table that rests my night ritual amenities and my phone. This table does not sit flat against the wall, allowing there to be a space. Constantly things fall behind the table, which have to be retrieved.

Solving this problem was very easy, all I did was attach a board to the back of table preventing items from falling. The idea for this came from looking at a box, which turned out to be the inspiration I needed to fix the problem.

When solving this problem, I learned that all it took was some inspiration. Being creative when solving problems is a key element in becoming a good coder. Getting ideas for solving problems can come from talking to fellow co-workers, research or just looking at examples and being inspired.

Tech explained in a Non-Tech way!

One of the easiest ways to group information together while coding is forming an object, which is a group of variables/values. But what do you do if you have several objects which are all pretty similar? You can put them into a list or object array.
Now that you have your object array you can perform different action to the specific groups or to the whole list of grouped objects. For example: there is an action called .push which will added another object to the end of your list.
Here are some more examples of array methods.

callback

These three array methods all invoke the callbackfunction

Rather than programmatically going through an array (eg. for loop) you can use methods inherited by an array to pass the values as parameters to a function. This type approach is called a callback.

.map

The method .map when applied to an array creates a new array with a new object name. It takes each element from the array and runs it through a provided function, which results in a new array.

.filter

The method .filter passes each value in the array to a function, which decides whether to filter out the value based on the logic within the function.

.reduce

The method .reduce reduces the individual values of an array to a single total number.

Example Syntax

var myArray = [1,2,3,4];
function myFunction (value){
  return value; //add logic to filter/map/reduce here
}
var newArray = myArray.filter(myFunction);
